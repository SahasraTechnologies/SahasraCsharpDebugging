⚠️ 10. Common Bugs in this Project

Students must identify:

❌ Duplicate ID allowed

👉 Should not allow

❌ Null crash
.First()

👉 Fix using:

FirstOrDefault()
❌ Wrong update logic
newFee = student.Fee;

👉 Should be:

student.Fee = newFee;
❌ Delete wrong student

👉 Always deleting first

❌ Divide by zero
total / count
